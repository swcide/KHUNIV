[KCU숭실사이버대학교 학생자모드 강의실 퀴즈 및 시험 설명서]

 

1.강의실에서 수강일람을 클릭하여 평가란에서 기간중인 “퀴즈” 혹은 “시험”을 클릭하여 참여합니다.

 

평가인정기간 : 시험에 참여할 수 있는 기간을 말합니다.

        (시험응시기간 이외에는 시험에 응시하지 못하오니 기간을 꼭 확인하여 주십시오)

 - 평가요소 : “퀴즈”, “시험”으로 표기되며 클릭하시면 응시할 수 있습니다.

 - 응시 : 응시 여부를 알 수 있습니다. (응시 : “O", 미응시 : "X", 진행중 : 표기안됨)

 - 결과 : 응시 후 평가인정기간이 만료되고 교수님의 평가가 마감되면 “완료”로 메시지가 표기되며, “완료”를 클릭하면 점수 및 피드백을 확인할 수 있습니다.

 

A 경우 : 퀴즈에 정상적으로 응시하고 평가결과가 공표된 경우

B 경우 : 퀴즈에 응시를 못하였을 경우

C 경우 : 퀴즈가 진행중이며, 아직 참여를 하지 않은 경우

D 경우 : 시험이 진행중이며, 아직 참여를 하지 않은 경우

 

 

2.시험메인페이지에서 퀴즈 및 시험에 대한 정보 및 주의사항을 확인 후 시험참여를 클릭합니다.

 

시험제목 : 등록된 시험의 제목

 - 시험내용 : 시험의 대한 간략한 내용.

 - 문제수 : 출제된 전체 문제수.

 - 시험기간 : 학생자가 실지적으로 시험에 응시 할수 있는 시간.

 - 실시시간 : 학생자가 시험에 참여할 수 있는 기간.

 - 문제유형 : 등록된 문제에 대한 유형.

 - 주의사항 : 시험전에 확인하여야 할 주의사항.

 

 

3.학생자가 시험을 응시할 수 있는 시간이 표시되며 좌측 상단에 표기되며, 이 시간은 항시 준수하여 시험에 응시하시기 바랍니다.

 

시간을 모두 사용하였을 경우에는 더 이상 퀴즈 및 시험에 응시 할 수 없으니 시험시간안에 모든 문제를 응시 할 수 있도록 하여 주십시오.

 - 시험시간이란?

   a. 해당 퀴즈 및 시험시간이 30분이 경우에는 30분, 60분인 경우에는 60분 동안만 퀴즈 및 시험을 응시할 수 있습니다.

   b. 퀴즈 및 시험기간은 참여할 수 있는 기간이며, 시간은 참여 후 실질적으로 응시할 수 있는 시간입니다.

 

   [예제]

   * 시험기간 : 2004년 4월1일 00:00분 ~ 2004년 4월3일 23:59분

   * 시험시간 : 60분

   - 상황1 -

   2004년 7월19일 23:00분부터 7월19일 23시:59분 안에 시험에 응시하였을 경우에는

   60분 동안 시험을 응시 할 수 있습니다.(권장)

   - 상황2 -

   23:20분 경에 시험에 응시하였을 경우에는 39분 동안만 시험에 응시 할수 있습니다.

   (상황2의 경우 21분을 손해를 보며, 시험기간이 마감되면 어떠한 방법으로도 시험에 재응시 불가능합니다)

 

 

4.객관식의 단일 선택형 및 다중선택형 문제의 경우 해당 되는 답안에 답을 체크(다답형의 경우 1개이상 체크가능)한 후 확인 버튼을 클릭하면 다음문제로 이동합니다.

 

객관식문제의 경우 해당되는 답안을 체크를 하면 되겠습니다.

 a. 현재시각 : 시험을 응시하는 서버의 시간을 표시합니다.

   (모든학생들에게 동일한 시간을 제공하기 위하여 서버시간으로 시험을 진행합니다)

 b. 남은시각 : 시험시간을 표기하며, 시험시간이 0초가 되면 자동 종료됩니다.

 c. 답안체크란 : 객관식 단일, 다중선택형의 경우에 답안을 표기하는 체크박스입니다.

                정답을 체크한 후 확인 버튼을 클릭하면 다음문제로 이동합니다.

    * 단일선택형 : 보기중 답안이 하나만 있는 경우입니다.

                  (1개 이상의 답안이 체크가 되지 않습니다)

    * 다중선택형 : 보기중 답안이 1개 이상의 경우입니다.

                  (답이 1개일수도 있고 보기 전부가 답일 수도 있습니다)

 d. 재검토란 : 문제를 응시할 때 답안이 아리송하여 후에 다시 한번 문제를 확인하여 답안을 수정하려 할 경우 체크하며, 체크를 하지 않아도 후에 답안수정은 가능합니다.

다만 재검토를 클릭하면 본 매뉴얼 7번 그림의 a번과 같이 검토할 문제를 쉽게 찾을 수 있습니다.

 

================= 시험에 첨부파일(음성파일)등이 있을 경우 =================

 

첨부파일을 클릭하면 해당 파일을 확인 할 수 있습니다.

 - 음성파일의 경우 (파일명 : wmv, wma, asf)

   : 음성파일인 경우 파일명을 클릭하면 원도우미디어 플레이어가 실행이 되며, 미디어 플레이어에서 음성이 재생되게 됩니다.

   * 본교 홈페이지에서 MyCampus >> UserLink 의 “퀴즈및시험테스트”에서 음성파일 테스트가 가능하오니 꼭 테스트 하여 주십시오.

 - 참고자료 : 교수자가 문제출제시 이미지를 첨부하였을 경우 나타나는 부분입니다.

 

 

5.주관식 단답형 문제의 경우 해당되는 답안을 정확히 기재한 후 확인 버튼을 클릭하면 다음문제로 이동합니다.

 

주관식 단답형 문제화면으로 위와 같은 경우 해당되는 답안에 정확한 답안을 입력합니다.

 - 답안 : 키보드 자판을 이용하여 정확한 답안을 입력하시면 되겠습니다.

   >> 답안을 입력시 문제를 잘 확인하여 답안의 입력 방법을 잘 확인합니다. 예로들면 “띄어쓰기 및 대소문자 구분등에 대한 정확한 표기를 하시오”라는 지시사항이 발생시 교수자가 원하는 답안만 정확히 입력하며, 부연설명등을 입력하지 않도록 합니다.

 

 

6.서술형 문제의 경우 해당되는 답안을 모두 기입한 후 확인 버튼을 클릭하면 다음문제를 응시할 수 있습니다.

 

주관식 서술형 문제의 경우 답안에 내용을 자세히 입력한 후 확인버튼을 클릭하면 됩니다.

 - 답안 : 키보드자판을 이용하여 서술할 내용을 입력합니다.

 - 임시저장 : 답안을 입력시에 임시저장버튼을 클릭하시면 현재까지 입력된 답안이 입력되게 되며, 계속 답안을 입력하실 수 있습니다.

   >> 서술형문제의 경우 답안이 장문임에 따라 답안입력에 많은 시간을 소요할 수 있고 그에 따른 불의의 사고(컴퓨터다운, 인터넷끊김현상)등으로 힘들게 입력시킨 답안이 입력중에 사라져서 처음부터 새롭게 입력 할 경우가 있습니다. 이에 답안 입력중에 한번씩 임시저장버튼을 클릭하면 그 시각까지 입력된 답안이 저장이 되어 안정성을 유지할 수 있습니다.

   * 임시저장버튼이란?

     클릭한 그 순간까지 입력시킨 답안을 안전하게 보관하는 기능입니다.

 

 

7.재검토 화면에서 문제를 클릭하여 해당문제를 확인하고, 검토을 마친 후 확인 버튼을 클릭하면 다음화면으로 이동합니다.

 

- 문제를 모두 응시하면 위 이미지와 같이 재검토 화면이 나타납니다.

 

  * 재검토(a) : 문제 응시중에 학생자가 재검토란에 체크(4번에 b문장 참조)를 하면 위 이미지와 같이 표시 됩니다.

    >>재검토 체크는 위 이미지와 같이 학생자가 구분하기를 편리하게 하기 위하여 체크 하는것이기 때문에 체크유무를 떠나서 모든 문제를 다시한번 검토하실 수 있습니다.

 

  * 번호 : 문제의 번호(순번)입니다.

  * 문제(b) : 검토하고자 하는 문제를 클릭하시면 문제창으로 이동하여 해당문제를 검토 및 수정하실 수 있습니다.

 

  * 자주하는 질문 *

  [질문1]"전 2번에 정답을 1번이라고 표기하였는데 자꾸 3번이 정답으로 나타납니다."

   => 이와같은 경우에 표기된 답안은 3번임을 말하는것이 아닙니다. 여기서의 "3"은 문제내용(이미지첨부방식문제)이며, "3"를 클릭하시면 다른 검토창과 같은 형식의 검토 페이지로 이동합니다.

 

  [질문2]"재검토창이 안나타나고 시험이 종료되었습니다."

   => 위 이미지 화면이 재검토 창입니다. 각각의 해당문제를 클릭하시면 문제를 검토 및 수정할 수 있는 창으로 이동하며, 확인버튼을 클릭하면 시험이 종료되어 더 이상 재검토를 하시지 못합니다.

 

  [질문3]"재검토란에 체크가 안되어 있어도 재검토가 가능한가요?"

   => 재검토 표기는 학생자들이 재검토 하실 문제를 알아보기 싶게 하기 위해서 표기 되는 것이며, 재검토창에서는 재검토란 체크와 관계없이 모든 문제를 검토하실 수 있습니다.

 

 

- 6번문제를 클릭하여 해당문제를 확인 후 다시 재검토 창으로 이동되면 해당 문제는 자동으로 재검토 체크란이 사라집니다.

 

 

8.문제응시를 완전히 마치셨으면 시험종료 창에서 확인을 클릭합니다.

 

- 재검토창에서 모든 검토를 마친 후 "확인" 버튼을 클릭하면 "시험을 종료하겠습니까"라는 안내창이 나타나며 안내창에서 "완료"를 클릭하시면 시험이 종료됩니다.

 

 

9.총 문제 수와 학생자가 답안을 입력한 문제수를 확인 후 확인버튼을 클릭하면 시험이 종료됩니다.

 

- 시험이 종료되면서 총문제 수와 응시한 문제수가 표기 됩니다.

 

 

10.수강일람 페이지에서 해당주차 시험의 응시란에 "O"로 표기되면 시험이 정상적으로 완료된 것입니다.

 

 

 

11. 퀴즈 및 시험기간이 만료된 후 교수자의 평가가 마감이 되면 평가란에 “완료” 버튼이 나타나며, 완료 버튼을 클릭시 본인의 점수 및 교수자의 피드백 내용이 보여집니다.

 

- 각문제에 대한 정답 및 답안설명등을 확인 하실 수 있습니다.

- 피드백부분을 이용하여 교수자의 메시지를 확인 하실 수 있습니다.

 

12. 수고하셨습니다.
